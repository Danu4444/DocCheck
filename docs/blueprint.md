# **App Name**: DocCheck

## Core Features:

- Doctor Sign-up and Verification: Doctors can sign up by providing their license details and email for verification via OTP. The system will authenticate the doctor's license to ensure validity.
- Patient Sign-up/Login: Patients can sign up/log in using their phone number with OTP verification, and register their family members using a single phone number. OTP via phone number for patient login.
- Doctor Availability Display: Display the availability of doctors based on location, hospital, and specialization.
- Nearest Hospital Locator: Locate the patient using location permission to find and display the nearest hospitals.
- Patient Medical History: Store and display patient medical history, including past doctors, dates, hospitals, and prescriptions. Patients can upload images of prescriptions.
- AI Chatbot Assistance: AI chatbot to answer patient queries about their problem and provide general medical information using its tool.
- Appointment Booking: Patients can book appointments by creating tokens. Doctors set a limit to the number of tokens, and patients receive live token status and notifications. Also there will be a call option provided by doctor to patient. The web application shows the token number of the patient. The patients will receive notification and mobile should vibrate to inform about the time to get checked up.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6) to evoke trust and health. Inspired by the healthcare setting, the choice is meant to suggest reliability.
- Background color: Light blue (#E3F2FD) for a clean, approachable feel, related in hue but significantly desaturated.
- Accent color: Green (#81C784) to highlight key actions and information, an analogous hue to blue that provides a fresh, reassuring feel.
- Font pairing: 'PT Sans' (sans-serif) for both headlines and body text. Chosen for its blend of modernity and readability.
- Use clear, universally recognized icons for navigation and features.
- Clean, organized layout with a focus on readability and easy navigation.
- Subtle transitions and feedback animations to enhance user experience.
